## What's Inside

<Accordion title="Frontend Application">
  Next.js web application for hotel reservations including rooms, restaurants, and event spaces. Built with React 19, TypeScript, and Supabase integration.
</Accordion>

<Accordion title="API Service">
  Lightweight API service built with srvx, providing backend functionality for the reservation platform.
</Accordion>

<Accordion title="Monorepo Architecture">
  Turborepo-based monorepo with shared packages for UI components, TypeScript configurations, and ESLint rules.
</Accordion>

<Accordion title="Container Infrastructure">
  Docker containerization with multi-stage builds, Docker Compose orchestration, and production-optimized images.
</Accordion>

<Accordion title="Kubernetes Deployment">
  Complete Kubernetes manifests and Helm charts for deploying to production clusters with ingress, services, and deployments.
</Accordion>

<Accordion title="Terraform Automation">
  Infrastructure as Code for provisioning Vercel projects and managing environment variables across deployments.
</Accordion>

## Getting Started

<Steps>
  <Step title="Clone the Repository">
    Get the source code from GitHub:
    ```bash
    git clone https://github.com/PavoAyala/proyecto-de-devops.git
    cd proyecto-de-devops
    ```
  </Step>
  
  <Step title="Set Up Environment">
    Configure your environment variables:
    ```bash
    cp .env.example .env
    # Edit .env with your Supabase credentials
    ```
  </Step>
  
  <Step title="Run Locally">
    Start the development environment:
    ```bash
    pnpm install
    pnpm dev
    ```
  </Step>
</Steps>
